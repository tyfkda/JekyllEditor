FrontendTemplate
================

Project template for frontend development:

* ES6 (JavaScript) + eslint
* HTML
* SASS (CSS)
* JS test: Jasmine + Karma


## Requirement

* node.js
* npm


## Set up

```
$ npm install
```


## Usage

### Build (one shot)

```
$ npm run build
```

* Files are generated in `public`
* You can see the page in http://localhost:3000/

### Build (automatically)

```
$ npm run watch
```

### Release

```
$ npm run release
```

* Files are generated in `release`

### Clean up

```
$ npm run clean
```
